# AI Engagement Predictor

## Project Overview
This project analyzes social media engagement data to evaluate platform performance, content type effectiveness, and sentiment impact.

## Tools Used
- Python
- Pandas
- Matplotlib
- Jupyter Notebook

## Key Insights
- Instagram generates the highest average engagement.
- Polls and videos outperform static content.
- Emotional content drives higher interaction.
- Twitter underperforms compared to other platforms.

## Business Recommendation
Prioritize interactive and video content. Reevaluate Twitter strategy.
